FAQ

What is Teewars Legacy? - It is teeworlds inspired game made with 2000s technologies in old opensource style.

Is it opensource? - Yes, sure.

It is a mod for Teeworlds? - No, it isnt.

Where I can to download this game? - Now you cant download this game because it now in development.

When it releases? - I cant guarantee, but I think it will be around mid 2027 - late 2027.

What programming language is Teewars Legacy written in? - Teewars Legacy written on C++ with C libraries (like stdio.h, stdlib, etc). Also in TWL using libraries like SDL 1.2, SDL_image 1.2, SDL_mixer 1.2.

How I can be contributor? - You can make pull requests on github, and add your features, or fix bugs in issue tracker.

What software is used to make TWL music? Who made music for TWL? - Music for TWL made in Fast Tracker II (in linux clone Milky tracker) by Mykyta Polishyk (ZabidenHTF/TW).
